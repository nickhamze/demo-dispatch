We have started serving a new house tea blend in the kitchen, on a trial
basis through the end of January. If it does not catch on, it will quietly
go away. So far it is catching on.

It is a black-tea base — a Ceylon and a small amount of an Assam, both
loose-leaf — with cinnamon Tomás toasts in the small skillet at the stove,
a few crushed cardamom pods, and a single allspice berry per pot. It is
sweet enough that you will not need sugar. It is meant for cold mornings.

The name we are using on the [Menu](/menu/) is "House blend, winter," which
is wordy, but every other name we tried turned out to be a name a tea
company already uses and has presumably already trademarked.

If you would like to take some home, there is a small tin in the
[Shop](/shop/) — see the [House tea, in a small tin](/shop-house-tea-tin/)
post for what is in it and how it brews.
