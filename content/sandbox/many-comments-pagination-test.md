> This is sample content for theme previews. (Sandbox post.)

A short post that hosts 53 threaded comments to exercise comment pagination
at the WordPress default of five comments per page. The post body is
intentionally brief - the interesting thing on this page is the comment
list below.

The comments are auto-generated by `scripts/build-wxr.js` from the template
in `content/comments/many-comments.yaml` and include at least one comment
of every variant the spec requires: a code snippet, a long URL, an
emoji-only reply, an `@mention` (as plain text), an inline image, a comment
by the post author, a multilingual reply, a pingback, a trackback, and one
comment in pending-moderation status.
