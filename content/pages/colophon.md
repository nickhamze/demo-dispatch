# Colophon

This site is running the **Demo Dispatch** sample dataset, version 1.0.0,
released 2026-04-20.

## Type

The dataset does not bundle a typeface. The active theme provides whatever
typography it considers correct. The dataset has been tested against
serif, sans-serif, and slab body text at sizes between 14px and 22px.

## Palette

Every illustration on the site uses one of the following six colours, with
no others permitted:

| Hex       | Name             | Role          |
| --------- | ---------------- | ------------- |
| `#F4EFE6` | Paper warm white | Background    |
| `#1B1B1F` | Ink near-black   | Linework      |
| `#B8533A` | Terracotta       | Accent 1      |
| `#2F6F6A` | Ocean teal       | Accent 2      |
| `#D6A24E` | Mustard ochre    | Accent 3      |
| `#7A8C6F` | Sage green       | Accent 4      |

Every individual image uses the paper background, the ink linework, and at
most two of the four accent colours.

## Imagery

All thirty illustrations were generated against a single locked prompt
template, then cropped from a single oversized master to four aspect
ratios (1:1, 4:5, 16:9, 21:9). The full art-direction spec, including the
prompt template and the per-image subject-to-palette pairing, is in the
dataset repository under `spec/`.
